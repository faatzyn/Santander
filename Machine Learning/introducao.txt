Como a IA é vista hoje?
    IA GERAL
        Sabe fazer tudo
    IA RESTRITA
        Sistema específico para uma coisa (Direção autônoma, reconhecimento facial,...) para pequenos problemas
    Aprendizado de Máquinas
        Ensinar um sistema a criar uma inteligência ou não baseada em alguns desafios


Teste de Touring
    Robô enganar/persuadir um humano numa determinada situação
    Testa a capacidade de um robô se comportar como uma humano


ML <> IA ?
    ML é necessário um método de aprendizado para criar/adquirir a parte inteligente do sistema;
    Quando o sistema consegue generalizar as situações, ele vira um IA;

Objetivo do ML
    Programar computadores para aprender determinado comportamento ou padrão automaticamento a partir de exemplos ou observações (DATASETS)
